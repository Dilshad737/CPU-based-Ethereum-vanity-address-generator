# Ethereum CPU Vanity Address Miner

🚀 Ultra-fast CPU-based Ethereum vanity address generator.

## Features
- Multi-core CPU usage for parallel processing
- Optimized Keccak hashing with the `pycryptodome` backend
- Easy customization of target prefixes
- Prints private key and address as soon as a match is found

## Requirements
- Python 3.10 or higher
- `coincurve` for fast elliptic curve operations
- `eth-hash[pycryptodome]` for Keccak hashing
